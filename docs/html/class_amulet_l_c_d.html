Detailed Description

A class used to manage the UART state machine between the Amulet display and Arduino.

◆ serialEvent()

void AmuletLCD::serialEvent ( )

Heartbeat function that manages the state machine with incoming serial data. This should be called periodically in your main loop or your own serialEvent() if supported by your board. https://www.arduino.cc/en/Tutorial/SerialEvent If arrays longer than the serial buffer are requested and you have blocking code, then consider peppering this command within your block Alternatively, call UART_State_Machine from a real UART interrupt (advanced topic, additional libraries may be required, plus you need to make UART_State_Machine public)
